# Project Scope

## Overview
A modern wealth financial website with portfolio management, chart visualization, and NAV asset display.

## Core Features
- Portfolio display and management
- Interactive financial charts
- NAV (Net Asset Value) asset tracking
- Responsive layout for mobile and desktop

## Tech Stack
- **Bundler**: Vite
- **Styling**: Tailwind CSS
- **Language**: Modern JavaScript (ES2020+)

## Design Principles
- Universal design — fully responsive, mobile-first
- Clean, modern financial UI aesthetic
- Accessible and performant
