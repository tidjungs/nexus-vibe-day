## Getting Started

1. Clone the repo:
   ```bash
   git clone git@github.com:tidjungs/nexus-vibe-day.git
   cd nexus-vibe-day
   ```

2. Navigate to your team's directory and start building:
   ```bash
   cd <your-team>/
   ```

3. Initialize your frontend project with your preferred stack (e.g. Vite + React, Next.js, Svelte, etc.)

## Guidelines

- Each team works within their own directory
- Use any frontend framework or library you like
- Commit early, commit often
- Have fun and vibe!
